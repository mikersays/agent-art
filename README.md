# Agent Art

An experimental exercise exploring how AI agents interpret and apply Bauhaus design principles to generate geometric art.

## The Experiment

Can an AI agent, given the rules of Bauhaus design — primary colors, geometric primitives, asymmetric balance, form follows function — produce compositions that meaningfully reflect those principles? This repo is a testbed for that question.

110 SVG compositions were generated by teams of parallel AI agents, each given a unique theme and the Bauhaus ruleset. No human curation or post-editing was applied. The results range from faithful interpretations to surprising departures — revealing how agents internalize (or fail to internalize) aesthetic constraints.

## How It Works

- Each piece is a self-contained 800×800 SVG using only geometric primitives
- Agents work in parallel teams of 5, each generating a batch of compositions
- A shared color palette and set of design principles guide the output
- No raster images, no external resources, no JavaScript — pure SVG geometry
